# Login System in PHP (iSecure)
================================
A simple working Login System in PHP with-
* Database connectivity
* User verification
* Password hashing
* Prohibiting duplicate user creation
* Usage of SESSION variables.
---
# Database 
* Database name - users
* Table name - users
* Columns - [ username, password, dt]
